# Аквапарк «Одеса» — мобільний застосунок (прототип)

Кликабельний прототип застосунку аквапарку: квитки, абонементи, оплата
(Apple Pay / Google Pay / картка), QR для турнікета, карта проїзду,
карта відвідувань (штампи), жива погода, UA/EN.

## Файли
- `index.html` — основний проєкт (редагуємо тут). Фото підтягуються з `photos/`.
- `aquapark-prototype.html` — той самий застосунок одним файлом (фото вшиті). Зручно надсилати у Telegram.
- `photos/` — фотографії: `hero.jpg` (головний екран), `g1–g4.jpg` (галерея), `login.jpg` (вхід).
- `server.py` — локальний сервер для перегляду (`python3 server.py`, потім http://127.0.0.1:8078).
